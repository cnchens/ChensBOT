JSON配置

本小节只会介绍部分JSON配置的方法，完整版请至进阶查看

打开config.json（在根目录中），有以下几行代码需要特别注意：

    "mdb_conn" : "mongodb://127.0.0.1:27017/", 

    "superusers" : [114514], 
    "cmd_start" : [""], 
    "host" : "127.0.0.1", 
    "port" : "32808", 
    "api_root" : "http://127.0.0.1:32808",

这几行的配置方法在下表中已经列出来了

键名称 值类型 值默认数据 备注
mdb_conn superusers cmd_start host port api_root
str set str str str str
mongodb://127.0.0.1:27017/ 空 空 127.0.0.1 32808 http://127.0.0.1:32808
